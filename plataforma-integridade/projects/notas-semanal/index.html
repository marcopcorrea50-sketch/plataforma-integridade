Gestão de Notas SAP por Prioridade

Análise e priorização de notas SAP para manutenção

📋 Descrição

Ferramenta para análise e gestão das notas SAP classificadas por prioridade. Permite acompanhar o status das notas, identificar criticidade, gerenciar a carteira de notas e apoiar a tomada de decisão na priorização dos serviços de manutenção.

Responsável
Marco Correa
Atualização
Semanal
Formato
Excel (.xlsm)
Área
Manutenção

🚀 Como usar

Baixe o arquivo .xlsm e abra no Microsoft Excel habilitando as macros. A planilha utiliza automação SAP para importar as notas diretamente do sistema.

Após importar, os dados são classificados automaticamente por prioridade, permitindo visualizar a carteira de notas por criticidade e status de atendimento. Somente Marco Correa realiza atualizações da planilha.
