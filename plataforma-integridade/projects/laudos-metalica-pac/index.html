Como usar os laudos
Os arquivos .xlsx devem ser abertos no Microsoft Excel com macros habilitadas para usar a funcionalidade de inserção automática de fotos (duplo clique nas áreas cinzas). Os campos de texto, quantitativos e classificações funcionam normalmente sem macros. Após preenchimento, salve o arquivo e gere o PDF diretamente pelo Excel (Arquivo → Exportar).
