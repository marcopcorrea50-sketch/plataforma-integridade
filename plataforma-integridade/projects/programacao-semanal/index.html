Análise de Programação Semanal

Controle e acompanhamento da programação semanal de manutenção

📋 Descrição

Ferramenta para análise e acompanhamento da programação semanal de manutenção. Permite visualizar o status das ordens de serviço, acompanhar indicadores de cumprimento de programação e identificar desvios para ação corretiva.

Responsável
Marco Correa
Atualização
Semanal
Formato
Excel (.xlsm)
Área
Manutenção

🚀 Como usar

Para visualização rápida: clique em "Visualizar online" — a planilha abre diretamente no navegador sem precisar baixar nada.

Para atualização completa: baixe o arquivo .xlsm, abra no Excel habilitando as macros, atualize os dados e salve. Somente Marco Correa realiza atualizações.
